Scoping is weird in JavaScript

  • How many times have you seen weird behavior when doing something similar to this: 
    for (var i = 0; i < 5; i++) {
  $('a#link_' + i).click(function() {
    console.log(i);
  });
};
  • Remember what happens? 
    5
5
5
5
5
  • Why? Because the anonymous click handler function does not get called while i is in scope. The interpreter looks for the value of i in function's local scope but can't find it. It searches elsewhere and finds the last value, which is 5
