More jQuery selector writing

Using jQuery.find() is fast

  • $('#mySelector').find('div.myClass');
  • jQuery.find() is used internally when searching on a context. So using a context will never be faster, and will most likely be slower than using jQuery.find() 
    $('div.myClass', '#mySelector'); // using context - wrong

$('#mySelector').find('div.myClass'); // using find - correct
  • If you need to search deep in the DOM it is probably smartest to use jQuery.find()
