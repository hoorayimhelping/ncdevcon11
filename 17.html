Caching Selectors

  • jQuery doesn't cache selectors for you
  • So doing this: 
    var height = $('#myId').height();
var width = $('#otherId').width();
var bgColor = $('#myId').css('background-color');
    is bad and inefficient mmmkay
  • Instead, cache the selector 
    var $myId = $('#myId');

var height = $myId.height();
var width = $('#otherId').width();
var bgColor = $myId.css('background-color');
  • This will save you an expensive DOM lookup
