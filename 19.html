Minimize DOM manipulation

  • Manipulating the DOM is mad expensive yo, try to keep it to a minimum. If you're going to do it, do it rightly instead of wrongly.
  • Wrongly: 
    var $myList = $('ul.myList');

for (var i = 1; i < 10; i++) {
	$myList.append('<li>I heard you like lists!! index: ' + i + '</li>');
}
  • Rightly: 
    var $myList = $('ul.myList');
var listHtml = '';

for (var i = 1; i < 10; i++) {
	listHtml += '<li>I heard you like lists!! index: ' + i + '</li>';
}

$myList.append(listHtml);
