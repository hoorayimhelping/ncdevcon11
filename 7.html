What do?

Use the === (strict equality) operator

Strict equality operators ===, !== do type and value checking

  '' === '0' // false
  0 === '' // false
  0 === '0' // false

  false === 'false' // false
  false === '0' // false
  false === 0 // false

  false === undefined // false
  false === null // false
  null === undefined // false
