More on Caching

  • Be aware, if you have a cached selector, and code somewhere else modifies its DOM element, your cached element will be dirty 
  • $myId = $('#myId');
$myId.attr('title', 'Dirty title');

// other code that got run:

$('a.link').live('click', function() {
  $('#myId').attr('title', 'New title');
});

console.log($myId.attr('title')); // Dirty title
